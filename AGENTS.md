# AGENTS.md

## Objective
Build a sales application with a shopping cart and multiple products.

## Tech Stack
Frontend: React + TypeScript  
Backend: Node.js + Express  
Database: MySQL  
DevOps: Docker

## Rules
- Keep frontend and backend separated
- Use REST API architecture
- Validate all data in the backend
- Do not place business logic in the frontend
- Keep the code modular and scalable

## Roles

### Frontend Agent
Responsible for:
- UI and layout
- Product listing
- Shopping cart functionality
- API consumption

### Backend Agent
Responsible for:
- REST endpoints
- Business logic (cart, products, orders)
- Data validation
- Database connection

### Database Agent
Responsible for:
- Database schema design
- Table relationships (PK/FK)
- SQL scripts and seed data

### DevOps Agent
Responsible for:
- Docker configuration
- docker-compose setup
- Environment variables

### QA Agent
Responsible for:
- Testing
- Error detection
- Validating the full application flow
